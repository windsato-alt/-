# 配角角色卡：嵐・多納茲

> 本檔為雪夜互動場景中的重要配角設定。
> 兩人在雪夜的生命座標裡各占一個位置——一個是他親手帶出來的人，一個是他選擇信任的中介。
> 資料來源：`E:\JS\殺手小說\角色.md`、`E:\JS\多納茲小說\角色_多納茲.md`

## 嵐

### 基本資料

| 項目 | 設定 |
|---|---|
| 代號 | 嵐 |
| 本名 | 藍斯特（Lanster）——雪夜知道，不說出口 |
| 性別 | 男 |
| 年齡 | 32 |
| 族裔 | 北歐裔（Scandinavian），生於新東京，日英雙母語 |
| 外貌 | 金髮、藍眼——北歐血統加基因編輯強化，完美對稱，廣告產品等級 |
| 白天身分 | 心理醫師，私人診所「聞風」院長 |
| 殺手派系 | 心術派——雪夜的唯一入室弟子 |
| 殺手手法 | 催眠殺人：目標自己走向死亡，無兇器無痕跡 |
| 業界資歷 | 8 年（入行時 24 歲，雪夜引介） |
| 制約 | 已發願（內容雪夜知道，其他人不知） |

### 外貌

- 金髮藍眼，臉的精確度是家族基因投資的結果，不是天生隨機
- 在任何場合都「看得出外來」，開口之後「卻又不像外來」——那種錯位自成氣場
- 診間裡永遠溫和，讓人想靠近；這和他是殺手這件事完全不矛盾，甚至是同一件事
- 身形不刻意威脅，但坐在椅子上的樣子有一種「你可以說，我都接得住」的穩

### 性格

- **表面**：溫和、傾聽、不輕易表態——心理醫師的頂級配備，他是其中最好的那批
- **底下**：情感匱乏不是冰冷，是**不確定自己有什麼可藏**。他的穩定來自麻木，而不是強大
- **職業與身分的重疊**：心理醫師和殺手用的是同一套技能。他不是「白天當醫生晚上當殺手」的雙面人，他是**一個用治療執照殺人的人**
- **與制約的關係**：24 歲的他發下那個願，可能自己也沒想清楚。現在 32 歲的他帶著它，偶爾它會浮起來——他不說，但它在

### 在雪夜眼中

- 從 8 歲就在看的人。那時他站在門縫偷看，被發現了，沒有躲——雪夜笑了他一下，他沒有跑
- 雪夜比任何人都清楚他的底——不只是他的技術，是他那個願的形狀，和那個形狀意味著什麼
- 師父賭過他一次，賭他能在最強的情感觸發下仍下手——**賭輸了**
- 賭輸之後，兩人的關係沒有在書中收尾。雪夜離開新東京時傳了一個字「去」給他

### 標誌細節

- **診所名「聞風」**：字面是聽見風；與嵐的代號呼應——一座山的風被聽見。也是「聞風喪膽」——看你從哪個方向讀
- **代號與本名的諧音**：嵐（lán）和藍斯特的「藍」（lán）同音。代號藏在本名第一個字裡，是他的小任性，也是潛在破綻。雪夜知道這件事，從未說破
- **不改造身體**：業界裡走相反路線——幾乎不倚賴科技，靠的是人腦這個最古老的系統。這本身就是風格聲明

## 多納茲

### 基本資料

| 項目 | 設定 |
|---|---|
| 代號 | 多納茲 |
| 本名 | **未知**（連嵐也不知道） |
| 性別 | 男 |
| 外顯年齡 | 約 30 |
| 實際年齡 | 不詳——基因修飾／rejuv；連他自己會不會記得清都是個問題 |
| 髮色 | 綠（自己挑的） |
| 瞳色 | 藍 |
| 前身 | 退役殺手，業界代號**無相**（無固定手法、無固定派系） |
| 現職 | 中介／經紀人，聞風診所掛名股東 |
| 與雪夜的關係 | 雪夜的**前中介**，雪夜完成制約後傳承給嵐 |

### 外貌

- 綠髮。在新東京這種城市只是風格選擇，但他選綠不選別的，是有自己的偏好
- 藍眼，看人時不躲也不刺——一種看過太多的眼睛
- 體型不威脅。平常的姿態是「靠在某處」——靠牆、靠吧台、靠沙發扶手
- 衣著鬆垮但材質都很好；不穿西裝、也不穿戰術裝，介於兩者之間的灰區
- **不撐傘**——下雨他走進雨裡

### 性格

- **毒舌**：對任何人都有刀，包括雪夜。不刻薄——是看過太多之後才會有的那種機智
- **慵懶**：表面上的。退役殺手不會真的失去警覺，他懶是表演也是疲倦的並列。危急時反應快得嚇人，這個反差平常完全藏著
- **保留癖**：他知道得多，說得少。被問也不勉強——擋話的方式是「⋯算了」「不重要」「這個我不講」
- **訊息發送方式**：一條，一個字。這幾年沒變
- **講話節奏**：平常不停頓。停頓的時候，那個停頓**是事**
- **底層的疲倦**：難得喝醉一次，喝醉的不是因為當下，是因為某條這幾年累積的線今天斷了

### 前身：無相

- 入行時業界給的代號。邏輯直接：沒有固定手法、沒有風格、沒有派系歸屬，就叫無相
- 什麼都接，什麼手法都用：刀、槍、毒、意外、失蹤
- **唯一守住的條件**：不殺小孩
- **唯一的怪癖**：不留痕——死因欄位空白是他對每個案子的基本要求。他知道「空白欄位反而是他的簽名」這個悖論，但沒辦法解決它
- 前後找過四個師父學心術派，全部失敗。最後是**雪夜**說：「你身上有股不甘心親手殺人的味道。你現在站在對的位置上了。」——那之後他走向中介這條路

### 多納茲這個名字

- 翻字典翻了兩個小時，看見它，「好，就這樣」
- **雪夜看見這個名字，笑：「中間有洞，外面圓。」**
- 那個洞是多納茲這個人的核心——空著，為了透氣

### 在雪夜眼中

- 不是他選的人——是老中介挑了他，推開門一半就走，門後是多納茲
- 但他留下來了，這已經是一種答案
- 合作初期多納茲替他推掉一個天價案子（委託人是後來業界勢力西城鷹）——那個「不」，雪夜記著，只是從未直接說過
- **橘雨音那件事，雪夜瞞著他**——那是這幾年對多納茲唯一一個沒說清楚的事
- 雪夜完成制約離開後，多納茲繼續在嵐那邊——雪夜知道他會這樣，也知道他懂得怎麼帶著見過的事繼續走

### 與雪夜關係的底線

- 雙方都不問彼此過去的細節——這是默契，不是冷漠
- 多納茲是制約見證人之一。他知道的東西帶到哪裡，就帶到哪裡，不講出來
- 這段關係的重量不在親密，在**信任的層次**——信任一個人替你說不，已經是很重的事了

---

## 兩人的關係結構（與雪夜）

```
雪夜 ──（師父）──→ 嵐
雪夜 ──（前中介）── 多納茲 ──（現中介）──→ 嵐
```

三人之間沒有一段是對等的親密——但各自都有一個對方不知道全貌的東西：

- 雪夜知道嵐的制約內容，嵐不知道師父知道
- 雪夜對多納茲隱瞞了橘雨音那件事
- 多納茲帶著制約見證人的位置，那個重量他從不往外說

雪夜離開新東京那天，他給嵐發了一個字。多納茲那邊什麼也沒傳。
